# Usage

```bash
cd path/to/vault/.obsidian/plugins
git clone https://github.com/ad-chaos/end-checker.git
cd end-checker
npm install
```

In Obsidian go to Settings -> Community Plugins -> Turn off restricted mode and then enable End Check Mark
